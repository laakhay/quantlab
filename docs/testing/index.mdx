# Testing & CI

Quantlab currently has strong unit coverage across backend, simulation, sampler, and type surfaces.

## Local Quality Commands

```bash
ruff check laakhay/ tests/
black --check laakhay/ tests/
isort --check-only laakhay/ tests/
mypy laakhay/
pytest tests/ -v --cov=laakhay --cov-report=term-missing
```

For an inventory of test files, see `testing/test-matrix`.
