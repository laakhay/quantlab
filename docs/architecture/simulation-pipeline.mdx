# Simulation Pipeline

## Samplers

`BaseSampler` establishes the probability interface:

- `sample`
- `pdf`, `cdf`, `ppf`
- `mgf` and derived moment properties (`skew`, `ex_kurtosis`)

`GaussianSampler` provides the concrete normal distribution implementation.

## GBM Flow

`GeometricBrownianMotion` path generation sequence:

1. `generate_shocks`: standard normal innovations.
2. `build_unit_paths`: cumulative Brownian paths.
3. `build_log_paths`: drift/diffusion transform in log space.
4. `scale_to_spot`: exponentiation to price space.

High-level entry points:

- `generate_paths`
- `terminal_values`
- optional antithetic variance reduction
