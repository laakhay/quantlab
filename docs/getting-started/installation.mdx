# Installation

## Base Package

```bash
pip install laakhay-quantlab
```

The base package avoids hard dependency on large array frameworks.

## Install With Backend Extras

```bash
pip install "laakhay-quantlab[numpy]"
pip install "laakhay-quantlab[jax]"
pip install "laakhay-quantlab[torch]"
pip install "laakhay-quantlab[all]"
```

## Verify Runtime Backends

```python
from laakhay.quantlab.backend import list_backends, has_backend

print('registered:', list_backends())
print('numpy:', has_backend('numpy'))
print('jax:', has_backend('jax'))
print('torch:', has_backend('torch'))
```

`fallback` may be present in `list_backends()`; it is a safety backend, not a compute backend.
